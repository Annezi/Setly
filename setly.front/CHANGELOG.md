# Changelog

Все заметные изменения проекта фиксируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
семантика версий — [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Added
- Единый компонент `ImageWithSkeleton` для плавной загрузки изображений карточек.
- Skeleton/fallback-состояния для ключевых экранов и секций (без пустых промежуточных рендеров).
- Точечные настройки `priority/fetchPriority/sizes` для above-the-fold изображений.

### Changed
- Карточки в разделах чек-планов, личного кабинета, статей, тестов и главных блоках переведены на более плавную загрузку изображений.
- Улучшены fallback-блоки для динамически загружаемых секций на главной и landing-странице.
- Устранены причины нестабильного dev-рендера/fast refresh (без потери функционала).

### Fixed
- Убраны блокирующие lint-ошибки, `yarn lint` проходит успешно.
- Снижены риски «ломающейся» верстки при hot reload в процессе разработки.

## Как вести changelog дальше

При каждом заметном изменении добавляйте записи в `## [Unreleased]`:
- `Added` — новая функциональность
- `Changed` — изменение существующего поведения/UX
- `Fixed` — исправления багов
- `Removed` — удаленный функционал (при необходимости)

Перед релизом переносите `Unreleased` в новую версию:

```md
## [1.4.0] - YYYY-MM-DD
### Added
- ...
```
